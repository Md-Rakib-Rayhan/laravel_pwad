Create database

Template Edit (custom)
----------------------
1. move dist(all library folder) folder to public folder
2. go to resources/views/auth/
			-> change the the name of login.blade.php (as a backup)
			-> create a new "login.blade.php" and paste the login page frontend code
			-> change the name of fields, give form action -> action="{{ route('login') }}"
			-> lastly give "@csrf" inside form
3. Change dashboard
			-> go to resources/views/ (here we found the current dashboard)
			-> create a folder and file named "backend/dashboard.blade.php"
			-> link the new file from "routes/web.php"
			->find this 
						-> Route::get('/dashboard', function () {
							return view('dashboard');
						})->middleware(['auth', 'verified'])->name('dashboard');
			-> Then Change link
			->  return view('dashboard');
				-> with
				return view('backend/dashboard');

			-> linking with layouts/component
				->  @include("backend.layouts.header")
				->  @yield("content")

				->  @extends ("backend.layouts.master")
				->  @section("content")
